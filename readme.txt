== Description ==
The fastest and easiest way to add meta fields to WordPress. Add unlimited meta fields to WordPress post types including posts and pages.

== Installation ==
= Manual =
1. Upload the Easy Meta Builder folder to the plugins directory in your WordPress installation
2. Activate the plugin.
3. Navigate to the \"Meta\" Menu.

= Admin Installer =
1. Visit the Add New plugin screen and search for \"Easy Meta Builder\"
2. Click the \"Install Now\" button.
3. Activate the plugin.
4. Navigate to the \"Meta\" Menu.
